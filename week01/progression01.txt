ให้ chatgpt สร้างโจทย์มาให้ทำ 5 ข้อ เกี่ยวกับเรื่อง array เพื่อให้เข้าใจให้มากขึ้น
ข้อ 1 สร้าง array เก็บสี 5 สี จากนั้นใช้ length เพื่อเช็คความยาวของ array แล้วเช็คสีตำแหน่งแรกและตำแหน่งสุดท้าย

ข้อ 2 สร้าง array ขึ้นมา แล้วเพิ่มข้อมูลเข้าไปใน array โดยใช้ .push()

ข้อ 3 ลบ array ตัวสุดท้ายโดยใช้ .pop()

ข้อ 4 ใช้ spread operater เปลี่ยนจาก string เป็น array ของตัวอักษรแต่ละตัว

ข้อ 5 สร้าง function ที่รับค่าเป็นแล้วแสดงแค่ตัวแรกกับตัวสุดท้าย
