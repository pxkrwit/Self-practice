ใช้ Chatgpt ในการอธิบายว่าโค้ดทำงานยังไง และได้ลองทำความเข้าใจโค้ดมากขึ้น
โค้ดที่เขียนคือ function ที่เขียนคือ function ที่ใช้เช็คว่าทั้ง 2 object นั้นมีค่าเท่ากันไหมโดยจะเช็คจาก key , value

โดยจะมีการเช็ครอบแรกคือเช็คทุกอย่างว่าทั้ง 2 ฝั่งเท่ากันไหม ถ้าใช่โปรแกรมจะหยุดทำงานและคืนค่า true
เช็ครอบ 2 จะทำการเช็คเอาแค่ type object และไม่เอาแค่ว่าง
และได้สร้างตัวแปรขึ้นว่าเพื่อเก็บ key ของ object
จากนั้นก็เอาตัวแปรที่เพิ่งสร้างขึ้นมาไปเช็คว่ามีความยาวเท่ากันไหม
ส่วนการเช็ค value จะใช้ for ... of ในการ loop object
