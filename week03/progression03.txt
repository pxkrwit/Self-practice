นำโจทย์ที่ยังไม่สามารถทำให้เสร็จได้ด้วยตัวเองภายในคลาสเรียนมาทำใหม่อีกครั้ง ทำให้เข้าใจการเขียนและวิธีการคิดมากขึ้น
โดย function นี้คือ function ที่เอาไว้เช็คจำนวนคำซ้ำในประโยคและคืนค่าเป็น object
function นี้จะรับแค่ string ถ้านอกกรณีจะคืนค่าเป็น undefined
โดยจะใช้ toLowerCase()ในการทำให้เป็นตัวเล็กทั้งหมด แล้วต่อด้วย .split(" ")ในการแยกคำให้เป็น array ด้วยช่องว่าง
สร้าง object ว่างเพื่อรอเก็บค่า
จากนั้นใช้ for ... of ในการ loop array โดยจะเช็คว่าถ้าหากเป็นคำซ้ำจะให้บวก 1 แต่ถ้าไม่ซ้ำจะให้ค่าเป็น 1
